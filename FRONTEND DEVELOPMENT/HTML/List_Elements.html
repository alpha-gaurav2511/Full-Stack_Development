LIST ELEMENTS

Lists are used to group related items together in a structured way.

HTML provides three types of lists: ordered lists, unordered lists, and definition lists.

1. Ordered Lists

It is denoted by "<>" and "ol" inside it.And then content inside "<>" and "li"

It lists by numbering(1,2,3,...).

2. Unordered Lists

It is denoted by "<>" and "ul" inside it.And then content inside "<>" and "li"

It lists by bullet point(dots).

3. Description List

It is denoted by "<>" and "dl" inside it.And then content inside "<>" and "dt" ."dt" is Description Term and then it's Description Details, which is denoted by "<>" and "dd" inside it.
