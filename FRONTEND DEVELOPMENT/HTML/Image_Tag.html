IMAGE ELEMENT

It is used to display images on web pages.

SYNTAX:-

<_img src="image address">

Example:- Image from online

Image from pc:-

BASIC SYNTAX:-

SYNTAX:-

<_img src="image.jpg" alt="Nature Image">

"src" && "alt" are attributes

  1. "src" = source
    • It is basically the image location
  2. "alt" = alternative text.
    • When the image cannot load it will show the content of alt i.e name of image

WIDTH & HEIGHT

SYNTAX

<_img src="cat.jpg" alt="Cat" width="300" height="200">
