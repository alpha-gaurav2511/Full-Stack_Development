Anchor Tags

Anchor tag is used to create links(hyperlinks) in HTML.

It helps users to:-

  • Open another website
  • Move to another page
  • Open email links
  • Download files

SYNTAX:-

<_a href="URL">Link Text<_ /a>

"href" = Hypertext Reference

Without "href" anchor tag will not work as a link

EXAMPLE:-

LINK TO CRICKET

To open the link in new tab we use target="_blank" after the href url.
