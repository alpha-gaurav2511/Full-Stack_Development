BoilerPlate

A boilerplate is a template that contains the basic structure and elements of a web page. It is used as a starting point for creating new web pages, and it can save time and effort by providing a pre-designed layout and code structure.

1.!DOCTYPE:-

!DOCTYPE html tells the browser that the document is an HTML5 document.

It is not an HTML element, but a declaration that informs the browser about the HTML version being used.

It must be the first line in an HTML document.

2. <'html'>

The <'html'> element is the root element of an HTML document. It contains all other elements in the document.

It is the parent of all other elements in the document.

"lang = "en"

This is an attribute of the <'html'> element.

It specifies the language of the document. In this case, it is English.

3. <'head'>

The <'head'> element contains meta information about the document, such as its title and character set.

It contains information ABOUT the webpage.This data is usually not displayed directly on the page.

There can be only one element in an HTML document.

# <_meta charset="UTF-8">

This is a meta element.

It specifies the character encoding for the HTML document. In this case, it is UTF-8.

UTF -8 Supports:- english,hindi,emojis and almost all symbols.

ViewPort Meta Tag

# <_meta name="viewport" content="width=device-width, initial-scale=1.0">

This is a meta element that is used to control the layout on mobile browsers. It sets the width of the viewport to the width of the device and sets the initial zoom level to 1.0.

Without this meta tag, mobile browsers may not display the page correctly.Mobile websites look zoomed out.Layouts break badly.

#<_title>

The <'title'> element defines the title of the document. It is displayed in the browser's title bar or tab.

4. <'body'>:-

The <'body'> element contains the content of the document that is displayed in the browser. It is the main part of the HTML document where you put all the visible content such as text, images, links, etc.
