"div" element is a generic Block Container.

  • starts from new line
  • takes full width
  • used for big sections/layouts

EXAMPLE

Hello, how are you??

I hope you are doing well

When you will inspect it then we can see that above ones are in one container.

span element

It is an INLINE CONTAINER.

  • starts from same line
  • takes only required width
  • used for small text/content

Hello,What's your name? Tell me

When we will inspect then we will see onlyname has separate box .
