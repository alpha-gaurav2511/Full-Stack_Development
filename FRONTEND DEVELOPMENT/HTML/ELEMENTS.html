ELEMENTS:-

An HTML element is a complete structure made using an opening tag(<_opening_tag>), content, and usually a closing tag().

1. HEADING ELEMENT:

Heading Element is given by <_>(angle brackets) and 'h' inside it and the number 1 to 6 is given to specify the level of heading.

The smaller the number the more important the heading is.HTML provides 6 headings levels.

2. PARAGRAPH ELEMENT:

Paragraph Element is given by <_>(angle brackets) and 'p' inside it. and it is used to define a paragraph in HTML.

It is a block-level element, which means that it starts on a new line and takes up the full width available.

3. BOLD ELEMENT:

Bold Element is given by <_>(angle brackets) and 'b' inside it. and it is used to make the text bold in HTML.

4. ITALIC ELEMENT:

Italic Element is given by (angle brackets) and 'i' inside it. and it is used to make the text italic in HTML.
