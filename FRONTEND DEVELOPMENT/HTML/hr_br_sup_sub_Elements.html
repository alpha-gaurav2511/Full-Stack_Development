1. hr Element

    2. Used for creating a horizontal line.

    Example:-

    Cricket is my favourite game.

    But i hate csk.

  2. br Element

    3. Used for breaking the line.

    Used to move content to next line.

    Example:-

    The morning breaks on Yamuna’s shore,
    With whispered winds from days of yore.
    The temples chime, the river flows,And Mathura in sunlight glows.
    • here br has been used after "shore," & "yore." so the content has moved to next line but not used in third line so its in the same line.

  3. sup Element

    4. It is used to write superscripts.

    Usually somewhat up the normal height.

    Example:-

    GauravSingh

    (a+b)2= a2 + b2+2ab

  4. sub Element

    5. It is used to write subscripts.

    Usually somewhat lower than the normal height.

    Example:-

    GauravSingh

    H2O

    C5H10
